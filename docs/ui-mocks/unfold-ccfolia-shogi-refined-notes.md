# UNFOLD CCFOLIA Shogi Refined Mock

対象ファイル:
- `docs/ui-mocks/unfold-ccfolia-shogi-refined.html`
- `docs/ui-mocks/unfold-ccfolia-shogi-refined.png`

## 何を洗練したか

1. 主盤面を卓のいちばん明るいエリアにして、最初の視線を中央へ集めた。
2. 左右の管理エリアを `recycle reserve / standby / deck` の3層に整理し、プレイ中に触る順で上下配置した。
3. 説明札はプレイ卓から外し、`manual scene` 側に逃がす前提でフッターのチップに格下げした。
4. 背景は既存の石目を活かしつつ、暖色の盤面とアイボリーのパネルで `UNFOLD` 本体の上品なUIに寄せた。
5. 赤青の識別は駒色だけでなく、パネルのトーンとバッジ色でも補強した。

## 反映の意図

- いまの配布用卓は `資料` と `プレイ` が同じ強さで並んで見えやすい。
- このモックでは `いま遊ぶ場所` と `周辺管理` の階層をまず立てている。
- 将棋モチーフは駒と盤面に任せ、卓UIそのものは `UNFOLD` 固有の落ち着いたプロダクト感を優先した。

## ココフォリア側に戻すときの要点

- 上段の説明札8枚は常設せず、資料シーンに分離する。
- 中央盤面は今より大きく、サイドエリアは静かに細くする。
- 再利用駒の置き場はサイド上段に固定し、山札/予備は下段へまとめる。
- ターン表示は盤面上部中央に寄せる。
- サイドの白い大型パネルは少しトーンを落とし、主盤面より目立たせない。
