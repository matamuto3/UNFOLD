プレイ中に見る面は、主盤面と両サイドの管理導線だけに絞ります。視線の強さと操作頻度を一致させるための構成です。

01
Main board is the brightest surface 中央盤面を卓全体の視線ハブにして、手番・読み・対戦状況が真っ先に入るようにする。
02
Recycle reserve stays near each player 再利用する駒は上段固定。手を戻す導線と最短距離に置いて、迷いを減らす。
03
Deck and standby sit one layer quieter 山札や待機駒は下段へ落として、触る頻度の低いものほど静かな面に寄せる。
