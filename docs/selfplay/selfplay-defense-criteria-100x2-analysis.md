# UNFOLD NPC守備評価 100戦検証メモ

生成日: 2026-05-04

条件: 先手=攻めNPC、後手=守りNPC、オリジナル駒100戦 + 将棋駒100戦、20戦ごとに分割。

注記: 今回は高速検証のため読みは `lookahead=1`。深読みではなく、評価関数にGPTプロ検討の守備基準を入れた状態の大量試行です。

## 参照したプロ検討の反映内容

- 即負け回避: 次手の王捕獲、本陣中央支配を強く避ける。
- 射線管理: 滅界者・角行・飛車・香車系の長射程ラインが王へ通る形を減点する。
- 逃げ道評価: 王の安全な逃げマスを加点し、0マスを大きく減点する。
- 遮断駒評価: 王筋を止める駒が自軍に守られている形を加点する。
- 踏み台警戒: 王筋の受け駒が取られて攻撃駒の着地点になる形を減点する。
- 跳躍警戒: 騎乗士・桂馬系の着地点が王周辺に入る形を減点する。
- 持駒防御: 王周辺に打てる自陣マスが残っている形を軽く加点する。

## 検討結果

### 1. 先手後手差はかなり縮んだ

オリジナル駒は先手勝ち51%、後手守備成功49%。将棋駒は先手勝ち49%、後手守備成功51%。「未決着を後手守備成功」と見るなら、目的だった先手一方有利の緩和はかなり近いところまで来ています。

ただし、後手の実勝利はオリジナル10%、将棋18%に留まっています。つまり、後手は守れるようになった一方で、守った後に勝ちへ反転する力はまだ弱いです。

### 2. 山札消費は狙いに近づいている

山札切れ対局はオリジナル47戦、将棋50戦。両者山札切れもオリジナル41戦、将棋44戦あります。勝敗バランスが寄りすぎず、自然に山札を使う長期戦が増えているので、現在の方向性は悪くありません。

ただし平均最終山札は先手2.6から2.7枚、後手4.2から4.3枚で、後手がやや山札を残しがちです。守備NPCは安全確認で手が止まり、展開・反撃に使い切れていない可能性があります。

### 3. まだ残る危険筋

オリジナルでは6手の本陣占領、将棋では4手の王捕獲が残っています。これは評価関数以前に、初期スタンバイ直後の形で「次の通常手番の攻撃路」が開きすぎるケースです。

特に将棋駒では、クロス/角行、ジェイ/飛車、ミラークランク/金将などが初期配置に絡むと、後手がティー/歩兵などで受けても王捕獲が間に合わない例が出ています。

### 4. 回収行動は長期戦に効いている

回収総数はオリジナル341、将棋411。長く守れた棋譜では、駒回収・展開図回収が単なる延命ではなく、攻撃路を切り直すために使われています。

次のNPC改善では、回収を「安全なら使う」ではなく、「相手の攻撃路を消す」「自分の反撃駒を再配置する」目的で評価するのがよさそうです。

## 次に強化する判断基準

1. 初期スタンバイ専用の安全検査を入れる。通常手番1手目で王捕獲・本陣占領が発生する形を、後手だけでなく先手も避ける評価にする。
2. 後手守備NPCに「守った後の反撃筋」を加点する。守備成功だけだと未決着に寄るため、相手王・本陣中央への反撃路を追加評価する。
3. 回収行動の目的を明確化する。駒回収は踏み台除去、展開図回収は攻撃路の切断として評価する。
4. 同一seedで評価関数のA/Bテストをする。今回の結果はかなり良いが、前回条件との差分を厳密に見るには同じseedで旧評価も回す必要がある。
5. 深読み版は20戦単位で試す。`lookahead=2` 以上はかなり重くなるので、100戦の前に20戦で品質と速度を確認する。
