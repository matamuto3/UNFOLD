条件: 先手=攻めNPC、後手=守りNPC、オリジナル駒100戦 + 将棋駒100戦、20戦ごとに分割。

## 追加した評価

- 相手王へ近づく配置・移動を加点。
- 相手王の安全な逃げマスを減らす局面を加点。
- 次手で王捕獲または本陣占領を狙える形を強く加点。
- 相手本陣中央への圧力を王捕獲に近い重要度で加点。
- 終盤ほど、回収・入れ替えよりも詰め筋を優先。
- 候補手の絞り込みに「次手勝ちを作る手」を追加。

## 検討結果

### 1. 未決着を消す目的は達成

前回はオリジナル39局、将棋33局が未決着でした。今回はどちらも未決着0です。寄せ・詰め評価を入れたことで、NPCは守るだけで止まらず、王捕獲または本陣占領へ向かうようになりました。

### 2. 勝率バランスも大きくは崩れていない

オリジナル駒は先手56%、後手44%。将棋駒は先手53%、後手47%。先手有利は残っていますが、後手も実勝利へ変換できるようになっています。

ただし、オリジナルの第5ブロックは先手13勝・後手7勝でやや先手寄りです。初期配置によって短手数の先手勝ちがまだ残っています。

### 3. 副作用として長期戦がかなり減った

平均手数はオリジナル31.9手、将棋29.6手。前回の77.1手、73.2手から大きく短くなりました。

山札切れ対局もオリジナル5局、将棋3局まで減っています。未決着は消えましたが、山札を使い切るような中長期戦はかなり減りました。

### 4. 回収行動がほぼ消えた

前回はオリジナル341回、将棋411回の回収がありました。今回はオリジナル0回、将棋2回です。

これは寄せ評価が強すぎて、回収を使って組み直す前に決着へ向かうためです。次の調整では「詰めがない局面では回収による再構築を許す」評価が必要です。

### 5. 決着理由は王捕獲が中心

オリジナルは王捕獲77、本陣占領23。将棋は王捕獲80、本陣占領20。寄せの主軸は王捕獲で、本陣占領は補助的な勝ち筋になっています。

本陣占領をもう少し増やしたい場合は、相手本陣中央に展開図を置く筋をさらに強化する必要があります。

## 次の調整案

1. 寄せ評価を少し弱める。未決着0は良いが、短手数決着が増えすぎています。
2. 詰め筋がないときだけ回収を復権させる。回収0に近いのは、ゲームの特徴を使えていません。
3. 初期スタンバイ直後の4手勝ちを専用に弾く。通常評価ではなく、初期配置の安全検査として処理した方がよさそうです。
4. 本陣占領ルートを少し増やす。現状は王捕獲が約8割なので、展開図ゲームらしい勝ち筋が少し弱いです。
5. 次は「未決着0を保ちながら平均50手前後」を目標にする。山札切れが10から25局くらい出る強さがちょうどよさそうです。
