# UNFOLD 現行ルール詳細と代表棋譜

このメモは、現行ブラウザ版の実装を基準にしたルール整理です。

- 基準実装: `laravel-app/public/app.js`
- ルール文: `laravel-app/public/rules.html`
- 代表棋譜の出典:
  - `docs/selfplay/selfplay-basepieces-original-depth3-8ply-probe.json`
  - `docs/selfplay/selfplay-basepieces-shogi-depth3-8ply-probe.json`

注意:

- リポジトリ直下の `app.js` と、古い `unfold-selfplay-*.json` には別系統の将棋駒対応が残っています。
- このメモでは、それらは使わず、`laravel-app/public` 系の現行ルールだけを採用しています。

## 1. 共通ルール

### 1-1. 盤面と本陣

- 盤面は `9 x 15`。
- 先手本陣は左端の `3 x 3`、後手本陣は右端の `3 x 3`。
- 本陣中央は先手が `(5, 2)`、後手が `(5, 14)`。
- 駒が移動できるのは、本陣または展開図で支配されているマスだけです。
- このゲームでいう `前` は相手本陣の方向です。
  - 先手は右向き
  - 後手は左向き

### 1-2. 開始時

- 各プレイヤーは山札 `20枚`、手札上限 `3枚`。
- 開始時に本陣中央へ王だけを置きます。
- その後、`初期スタンバイ` を行います。

### 1-3. 初期スタンバイ

- 先手後手交互に、手札から `3枚ずつ` カードを選びます。
- 初期スタンバイでは展開図を盤面に置きません。
- 選んだカードの対応駒だけを、自分の本陣 `3×3マス` の空きマスに `1体` 置きます。
- 選んだカードの展開図は手札ではなく `持ち展開図` になります。
- 対応駒3体を配置し終えた側は山札から補充し、手札を `3枚` に戻します。
- 両者が3枚ずつ終えると、先手の通常手番から開始します。

### 1-4. 1手でできること

通常手番では、次の `6種類` から `1つ` 選びます。

1. 展開図を置く
2. 駒を移動する
3. 持ち駒を打つ
4. 手札を入れ替える
5. 駒を回収する
6. 展開図を回収する

### 1-5. 展開図を置く

- 手札または持ち展開図を `0 / 90 / 180 / 270度` に回転して置けます。
- 置く展開図は、自分の現在の陣地に `一辺で接している` 必要があります。
- 自分がすでに支配しているマスの上には置けません。
- 相手駒がいるマスの上にも置けません。
- 相手の展開図の上に重ねることはできます。
  - ただし、そのマスに相手駒がいる場合は不可です。
- 手札から展開図を置いたら、その展開図の空きマス `1つ` に対応駒を `1体` 置きます。
- 展開図を手札から使うと、山札から補充して手札を `3枚` に戻します。
- 展開図を持ち展開図から使った場合、対応駒は置かず、手札も補充しません。

### 1-6. 駒の移動と捕獲

- 駒は、本陣や展開図で支配されているマスの上だけを移動できます。
- 相手の陣地上にも移動できます。
- 2マス以上進む動きや長距離直進は、途中のマスも連続して移動可能で、途中に駒がいてはいけません。
- 移動先に相手駒がいれば捕獲します。
- 相手の王を捕獲した時点で即勝利です。
- 王以外を捕獲した場合、その駒は自分の `持ち駒` になります。

### 1-7. 持ち駒

- 持ち駒は、自分が現在支配している空きマスに打てます。
- 本陣にも打てます。
- 相手が支配しているマスには打てません。

### 1-8. 手札入れ替え

- 手札を全て山札へ戻してシャッフルし、新しく `3枚` 引き直します。

### 1-9. 駒回収

- 自分の駒だけを回収できます。
- 回収できるのは `自分が支配しているマス` にいる駒だけです。
- 王は回収できません。
- 回収した駒は持ち駒に戻ります。

### 1-10. 展開図回収

- 手札枚数に関係なく行えます。
- 回収できるのは、次の条件を全て満たす `自分の展開図` です。
  - その展開図の上に駒がいない
  - さらに上に別の展開図が重なっていない
  - つまり、いま盤面の最上面に見えている
- 回収した展開図は手札ではなく `持ち展開図` になります。
- 持ち展開図は以後の手番で盤面に再設置できますが、対応駒は紐づきません。

### 1-11. 勝利条件

勝利条件は `2つ` です。

1. 相手の王を捕獲する
2. 相手本陣中央を自分の支配下にして手番を終える

補足:

- 2つ目の勝利理由は実装上の表示名が `本陣占領` です。
- 判定は各手番の終了時に行われます。

## 2. original 駒モード

### 2-1. 山札20枚の内訳

- 混沌獣 x1
- 護衛士 x2
- 結界士 x2
- 突撃士 x2
- 前衛士 x2
- 滅界者 x1
- 騎乗士 x2
- 界騎士 x2
- 側撃士 x2
- 攪乱士 x2
- 誘引士 x2

王は山札には入らず、開始時に本陣中央へ置かれます。

### 2-2. 展開図と対応駒

- `net01 ティー -> 混沌獣`
- `net02 / net02m クランク / ミラークランク -> 護衛士`
- `net03 / net03m サイス / ミラーサイス -> 結界士`
- `net04 / net04m ジェイ / ミラージェイ -> 突撃士`
- `net05 / net05m ジグザグ / ミラージグザグ -> 前衛士`
- `net06 クロス -> 滅界者`
- `net07 / net07m スピア / ミラースピア -> 騎乗士`
- `net08 / net08 ソー / ソー -> 界騎士 x2`
- `net09 / net09m フック / ミラーフック -> 側撃士`
- `net10 / net10m ステップ / ミラーステップ -> 攪乱士`
- `net11 / net11m ブロック / ミラーブロック -> 誘引士`

### 2-3. 駒の動き

- 展界者: 全方向へ1マス
- 誘引士: 後方へ1から2マス
- 側撃士: 上下1マス、前斜め2方向へ1マス
- 護衛士: 上下1マス、後方1マス、後斜め2方向へ1マス
- 前衛士: 縦横へ1マス
- 騎乗士: 前2マス、上下2マスへ跳躍
- 突撃士: 前方へ1から3マス直進
- 攪乱士: 斜め1マス、後斜め2方向へ2マス
- 結界士: 前後1マス、斜め4方向へ1マス
- 界騎士: 前1から2マス、前斜め2方向へ1マス、上下1マス
- 混沌獣: 縦横1マス、加えて `(上下1, 前後2)` の4方向へ跳躍
- 滅界者: 前方と前斜め2方向へ、連続した陣地上を何マスでも進む

### 2-4. 代表棋譜

出典:

- `docs/selfplay/selfplay-basepieces-original-depth3-8ply-probe.json`
- `game id: 1`
- `initialStandbyRule: basePieces`
- `reason: 未決着`
- `turns: 2`
- `plies: 8`

手順:

1. 先手 初期駒 攪乱士 `(5, 3)`
2. 後手 初期駒 護衛士 `(5, 13)`
3. 先手 初期駒 滅界者 `(4, 3)`
4. 後手 初期駒 前衛士 `(4, 13)`
5. 先手 初期駒 結界士 `(6, 3)`
6. 後手 初期駒 誘引士 `(6, 13)`
7. 先手 持ち展開図からクロスを展開
8. 後手 持ち展開図からミラークランクを展開

## 3. shogi 駒モード

### 3-1. 山札20枚の内訳

- 歩兵 x9
- 金将 x3
- 銀将 x2
- 桂馬 x2
- 香車 x2
- 飛車 x1
- 角行 x1

王将は山札には入らず、開始時に本陣中央へ置かれます。

重要:

- 現行 shogi モードは `通常将棋そのもの` ではありません。
- 駒の動きは将棋風ですが、山札20枚の構成は `UNFOLD 専用` です。
- `成り` はありません。
- 現行通常デッキには `と金 / 成桂 / 龍王 / 龍馬` を置くカードは含まれていません。

### 3-2. 展開図と対応駒

- `net01 ティー -> 歩兵`
- `net02 / net02m クランク / ミラークランク -> 金将`
- `net03 / net03m サイス / ミラーサイス -> 銀将`
- `net04 ジェイ -> 飛車`
- `net04m ミラージェイ -> 歩兵`
- `net05 / net05m ジグザグ / ミラージグザグ -> 歩兵`
- `net06 クロス -> 角行`
- `net07 / net07m スピア / ミラースピア -> 桂馬`
- `net08 ソー -> 金将 x1 と 歩兵 x1`
- `net09 / net09m フック / ミラーフック -> 香車`
- `net10 / net10m ステップ / ミラーステップ -> 歩兵`
- `net11 / net11m ブロック / ミラーブロック -> 歩兵`

### 3-3. 駒の動き

- 王将: 全方向へ1マス
- 香車: 前方に何マスでも直進
- 桂馬: 前へ2、左右へ1の跳躍
- 銀将: 前1マス、前斜め2方向、後斜め2方向へ1マス
- 金将: 前、横、後ろ、前斜めへ1マス
- 歩兵: 前へ1マス
- 飛車: 縦横に何マスでも移動
- 角行: 斜めに何マスでも移動

出典:

- `plies: 8`

手順:

1. 先手 初期駒 角行 `(5, 3)`
2. 後手 初期駒 金将 `(5, 13)`
3. 先手 初期駒 銀将 `(4, 3)`
4. 後手 初期駒 歩兵 `(4, 13)`

8. 後手 持ち展開図からジグザグを展開

## 4. ひとことで言うと

- 共通骨格は `展開図で陣地を伸ばし、その上で駒を運用するゲーム`
- `original` は UNFOLD 固有の駒性能セット
- `shogi` は 将棋風の駒性能セット
- どちらも `王捕獲` と `本陣占領` が勝ち筋
- `shogi` も通常将棋の完全再現ではなく、現行コード上は `UNFOLD 用の変形20枚デッキ` です
